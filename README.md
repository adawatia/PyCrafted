# 🚀 PyCrafted - A Collection of Mini-Projects

Welcome to **PyCrafted**, a collection of Python mini-projects designed to enhance problem-solving skills, logic, and programming expertise. Each project covers different domains, from games and simulations to algorithms and data compression.

## 📂 Project Categories

### 🎮 Games & Simulations

| **Project Name**      | **Description**                                                                                                                                                                    | **Key Features/Skills**                                                                                      | **Reference**                                |
|-----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|---------------------------------------------|
| **Bagels**            | A deductive logic game where players guess a secret 3-digit number based on feedback (like "1 digit correct"). It sharpens reasoning and pattern recognition skills.            | - Deductive reasoning<br>- User input handling<br>- Python logic implementation<br>- Looping & conditionals | [Link to Repository](Bagels)           |
| **PaperCuts**         | A colorful Rock, Paper, Scissors game where users play against the computer in a visually engaging way.                                        | - Game design<br>- Randomized computer choice<br>- Looping & conditionals<br>- Colored terminal output      | [Link to Repository](PaperCuts)            |
